*DNA GC Content Calculator*🧬
A robust Python command-line tool to process DNA sequences, handle FASTA formatting, and calculate the GC Content percentage.

🚀 _Features_

FASTA Parsing: Automatically detects and skips header lines (lines starting with >).

Sequence Normalization: Cleans whitespace, removes line breaks, and converts all bases to uppercase.

Strict Validation: Ensures the sequence only contains valid DNA bases (A, T, G, C). If an invalid character is found, the tool reports exactly what it is.

Accurate Metrics: Provides sequence length, total G+C count, and the final GC percentage.
